## Задание 1 — JSONPlaceholder (/posts)
Скрипт `api_posts.py` выполняет GET-запрос к публичному API JSONPlaceholder и выводит заголовки и тела первых 5 постов.

## Задание 2 — OpenWeather API
Скрипт `weather.py` принимает название города от пользователя, отправляет GET-запрос к OpenWeather API и выводит текущую температуру и описание погоды.

⚠️ Перед запуском `weather.py` необходимо указать свой API ключ OpenWeather в переменной `API_KEY`.git status
